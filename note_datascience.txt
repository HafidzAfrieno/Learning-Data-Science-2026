======================================================================
               CATATAN ENSEMBLE LEARNING & TREE MODEL
======================================================================

----------------------------------------------------------------------
A. METODE ENSEMBLE (Bagging vs Boosting vs Stacking)
----------------------------------------------------------------------

1. Bagging (Bootstrap Aggregating)
   - Konsep : Menggunakan model yang SAMA, dilatih secara PARALEL 
              pada subset data yang diambil secara acak.
   - Hasil  : Penggabungan prediksi menggunakan sistem Voting atau 
              Rata-rata.
   - Fokus  : Mengurangi Overfitting / Variance (Membuat model lebih
              stabil dan konsisten).

2. Boosting
   - Konsep : Menggunakan model yang SAMA, dilatih secara SEKUANSIAL 
              (berurutan/bergantian).
   - Kerja  : Model baru fokus belajar dari kesalahan/error yang 
              dibuat oleh model sebelumnya.
   - Fokus  : Menaikkan Akurasi / Mengurangi Bias (Membuat model yang 
              lemah menjadi kuat).

3. Stacking (Stacked Generalization)
   - Konsep : Menggunakan model yang BERBEDA JENIS, dilatih secara 
              BERLAPIS (layered).
   - Kerja  : Model di lapisan atas (meta-model) belajar dari hasil 
              prediksi kumpulan model di lapisan bawahnya.
   - Fokus  : Kombinasi kekuatan antar berbagai algoritma yang berbeda.


----------------------------------------------------------------------
B. KARAKTERISTIK MODEL BERBASIS TREE (POHON)
----------------------------------------------------------------------

* Fitur Scaling (Normalisasi / Standarisasi):
  - Aturan  : DATA X TIDAK PERLU DI-NORMALISASI ATAU DI-STANDARISASI.
  - Alasan  : Algoritma berbasis Tree (seperti Decision Tree, Random 
              Forest, XGBoost) mengambil keputusan berdasarkan batas 
              nilai (split/percabangan), BUKAN berdasarkan perhitungan 
              jarak geometris antar data (seperti KNN, SVM, atau 
              K-Means).





======================================================================
                  RAPOR EVALUASI MODEL MACHINE LEARNING
======================================================================

----------------------------------------------------------------------
A. METRIK REGRESI (Prediksi Angka/Nilai)
----------------------------------------------------------------------

1. MAE (Mean Absolute Error)
   - Fungsi Utama : Mengetahui rata-rata kesalahan tebakan model.
   - Karakteristik: Nilainya jujur apa adanya dan satuannya sama dengan 
                    data asli (misal: Rupiah, Kg, Meter).

2. MSE & RMSE (Mean Squared & Root Mean Squared Error)
   - Fungsi Utama : Menjadi "alarm" untuk kesalahan tebakan yang fatal.
   - Karakteristik: Memberikan hukuman/penalti berat jika model membuat
                    tebakan yang meleset terlalu jauh (ekstrem).

3. MAPE (Mean Absolute Percentage Error)
   - Fungsi Utama : Mengetahui persentase rata-rata kesalahan model.
   - Karakteristik: Paling mudah dijelaskan ke orang awam karena berupa 
                    persentase (misal: "Error model kita cuma 0.71%").

4. R2 & Adjusted R2 Score
   - Fungsi Utama : Mengukur seberapa pintar model memahami pola data.
   - Karakteristik: Skala 0 sampai 1. Semakin mendekati 1, model hampir 
                    sempurna menguasai variasi pada data.


----------------------------------------------------------------------
B. METRIK KLASIFIKASI (Prediksi Kategori/Label)
----------------------------------------------------------------------

1. Accuracy (Akurasi)
   - Fungsi Utama : Mengukur persentase total tebakan benar (Iya & Tidak).
   - Catatan      : Hanya bagus dipakai jika jumlah data antar kelas 
                    seimbang (50:50).

2. Precision (Presisi)
   - Fungsi Utama : Meminimalkan "Salah Tuduh" (False Positive).
   - Cocok Untuk  : Kasus yang tidak boleh salah tebak (misal: filter 
                    spam, rekomendasi investasi).

3. Recall (Sensitivitas)
   - Fungsi Utama : Meminimalkan "Target Lolos" (False Negative).
   - Cocok Untuk  : Kasus berbahaya yang targetnya tidak boleh lolos 
                    sama sekali (misal: deteksi kanker, deteksi bom).

4. F1-Score
   - Fungsi Utama : Mencari titik keseimbangan antara Precision & Recall.
   - Cocok Untuk  : Kasus di mana data antar kelas sangat timpang 
                    (misal: data transaksi normal vs penipuan bank).

5. ROC-AUC
   - Fungsi Utama : Mengukur kemampuan model memisahkan dua kelompok.
   - Karakteristik: Nilai 1 berarti model sempurna membedakan mana kelas 
                    A dan mana kelas B. Nilai 0.5 sama dengan tebak koin.

6. Log Loss
   - Fungsi Utama : Mengukur tingkat kepastian/keyakinan prediksi model.
   - Karakteristik: Nilai LEBIH KECIL = LEBIH BAGUS. Menghukum keras 
                    model yang "sok tahu" (sangat yakin tapi salah).

Dalam ekosistem Python, sebagian besar algoritma regresi populer diimplementasikan melalui dua *library* utama: **Scikit-Learn** (untuk metode statistik tradisional, *tree-based*, dan *ensemble*) serta *library* khusus seperti **XGBoost, LightGBM, CatBoost**, dan **PyTorch/TensorFlow** (untuk *Deep Learning*).

Berikut adalah **30 algoritma regresi paling populer di Python** yang dikelompokkan berdasarkan kemiripan cara kerjanya, lengkap dengan nama *class* atau modul aslinya di Python agar Anda bisa langsung mengimpornya:

## 1. Pendekatan Linear Tradisional & Regulasi (Scikit-Learn)

Algoritma yang mengasumsikan hubungan linear antara fitur dan target, dilengkapi teknik regulasi untuk mencegah *overfitting* pada data Python Anda.

 * **Linear Regression (Ordinary Least Squares):** Modul standar untuk regresi linear tanpa penalti.

   from sklearn.linear_model import LinearRegression

 * **Ridge Regression:** Regresi linear dengan regulasi L2 untuk mengatasi multikolinearitas.

   from sklearn.linear_model import Ridge

 * **Lasso Regression:** Regulasi L1 yang bisa menyusutkan koefisien fitur tidak penting menjadi tepat nol (seleksi fitur otomatis).

   from sklearn.linear_model import Lasso

 * **ElasticNet Regression:** Menggabungkan penalti L1 dan L2 secara bersamaan.

   from sklearn.linear_model import ElasticNet

 * **Least Angle Regression (LARS):** Sangat efisien untuk data Python yang memiliki dimensi fitur sangat tinggi (p >> n).

   from sklearn.linear_model import LARS

 * **Orthogonal Matching Pursuit (OMP):** Algoritma maju (*forward*) untuk memilih representasi fitur linear yang *sparse*.

   from sklearn.linear_model import OrthogonalMatchingPursuit

## 2. Berbasis Pohon Keputusan & Ensemble Tradisional (Scikit-Learn)

Algoritma non-linear berbasis pohon yang sangat fleksibel dan tidak memerlukan penskalaan fitur (*feature scaling*).

 * **Decision Tree Regressor:** Memotong ruang data menjadi segmen-segmen keputusan berbentuk hierarki.

   from sklearn.tree import DecisionTreeRegressor

 * **Random Forest Regressor:** Algoritma *bagging* yang menggabungkan prediksi dari ratusan pohon keputusan acak.

   from sklearn.ensemble import RandomForestRegressor

 * **Extra Trees Regressor:** Mirip Random Forest, tetapi memilih titik potong (*split*) secara acak untuk mengurangi varians model.

   from sklearn.ensemble import ExtraTreesRegressor

 * **Gradient Boosting Regressor:** Membangun pohon keputusan secara sekuensial untuk meminimalkan *error* dari pohon sebelumnya.

   from sklearn.ensemble import GradientBoostingRegressor

 * **Histogram-Based Gradient Boosting:** Versi cepat Scikit-Learn yang mengelompokkan data kontinu ke dalam bin integer (terinspirasi dari LightGBM).

   from sklearn.ensemble import HistGradientBoostingRegressor

 * **AdaBoost Regressor:** Fokus memperbaiki prediksi pada sampel data yang memiliki *error* tinggi di iterasi sebelumnya.

   from sklearn.ensemble import AdaBoostRegressor

## 3. Library Gradient Boosting Eksternal (Sangat Populer di Kaggle)

Tiga algoritma *state-of-the-art* yang memiliki *library* Python terpisah dan sering memenangkan kompetisi data science.

 * **XGBoost Regressor:** Implementasi *gradient boosting* yang sangat dioptimalkan, cepat, dan mendukung paralelisasi sistem.

   from xgboost import XGBRegressor

 * **LightGBM Regressor:** Varian berbasis histogram dari Microsoft, sangat cepat dan hemat memori untuk dataset skala besar.

   from lightgbm import LGBMRegressor

 * **CatBoost Regressor:** Algoritma dari Yandex yang dirancang khusus untuk menangani data kategorikal secara otomatis tanpa *one-hot encoding*.

   from catboost import CatBoostRegressor

## 4. Berbasis Jarak, Tetangga, & Kernel (Scikit-Learn)

Algoritma yang memprediksi nilai berdasarkan kesamaan jarak geometri antar data atau proyeksi ruang dimensi.

 * **K-Nearest Neighbors (KNN) Regressor:** Memprediksi nilai target berdasarkan rata-rata nilai dari *k* tetangga terdekatnya.

   from sklearn.neighbors import KNeighborsRegressor

 * **Radius Neighbors Regressor:** Memprediksi nilai berdasarkan semua titik data yang berada di dalam radius jarak tertentu.

   from sklearn.neighbors import RadiusNeighborsRegressor

 * **Support Vector Regression (SVR):** Menggunakan prinsip SVM untuk menemukan fungsi pembatas dengan margin toleransi kesalahan (\epsilon).

   from sklearn.svm import SVR

 * **Linear SVR:** Implementasi SVR yang dikhususkan menggunakan kernel linear, jauh lebih cepat untuk data baris besar.

   from sklearn.svm import LinearSVR

 * **Kernel Ridge Regression (KRR):** Memadukan regulasi Ridge dengan *kernel trick* untuk pemodelan non-linear.

   from sklearn.kernel_ridge import KernelRidge

 * **Gaussian Process Regressor (GPR):** Pendekatan probabilistik yang menghasilkan estimasi nilai beserta rentang ketidakpastian (*uncertainty interval*).

   from sklearn.gaussian_process import GaussianProcessRegressor

## 5. Pendekatan Robust / Tahan Outliers (Scikit-Learn)

Jika data Python Anda kotor dan dipenuhi banyak pencilan (*outliers*), kelompok algoritma ini adalah pilihan terbaik.

 * **Huber Regressor:** Menggunakan fungsi kerugian Huber yang tidak sensitif terhadap *outliers* besar.

   from sklearn.linear_model import HuberRegressor

 * **RANSAC Regressor:** Secara iteratif memisahkan data menjadi *inliers* dan *outliers*, lalu melatih model hanya pada data normal.

   from sklearn.linear_model import RANSACRegressor

 * **Theil-Sen Estimator:** Menghitung parameter regresi berdasarkan nilai median dari kemiringan (*slope*) pasangan data.

   from sklearn.linear_model import TheilSenRegressor

## 6. Pendekatan Bayesian & Stokastik (Scikit-Learn)

Algoritma regresi yang menggunakan pendekatan probabilitas untuk menentukan parameter atau dioptimalkan untuk data masif.

 * **Bayesian Ridge:** Memasukkan parameter regularisasi langsung ke dalam distribusi probabilitas Bayesian.

   from sklearn.linear_model import BayesianRidge

 * **ARD Regression (Automatic Relevance Determination):** Menghasilkan koefisien yang lebih *sparse* (banyak nol) dibanding Bayesian Ridge.

   from sklearn.linear_model import ARDRegression

 * **SGD Regressor:** Menggunakan *Stochastic Gradient Descent*, sangat cocok untuk *online learning* atau data yang terlalu besar untuk masuk RAM.

   from sklearn.linear_model import SGDRegressor

## 7. Jaringan Saraf & Transformasi Dimensi

Algoritma yang memanfaatkan arsitektur berlapis (*Deep Learning*) atau melakukan reduksi dimensi sebelum regresi dilakukan.

 * **Multi-Layer Perceptron (MLP) Regressor:** Jaringan saraf tiruan (*Neural Network*) standar di Scikit-Learn untuk hubungan non-linear kompleks.

   from sklearn.neural_network import MLPRegressor

 * **Partial Least Squares (PLS) Regression:** Menemukan komponen baru yang memaksimalkan kovarians antara fitur input dan target (mirip gabungan PCA dan Regresi).

   from sklearn.cross_decomposition import PLSRegression

 * **Quantile Regressor:** Alih-alih memprediksi rata-rata (*mean*), algoritma ini digunakan untuk memprediksi nilai kuantil/persentil tertentu dari data target.

   from sklearn.linear_model import QuantileRegressor
